 ## For Running the code

 1. Fork it
 2. Once forked, copy your repo's forked https/ssh url
 3. git clone 'the repo url'
 4. after changing the code : - git add .
 5. git commit -m 'message'
 6. git push origin main
 7. Create Pull Request
